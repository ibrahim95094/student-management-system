# Student Management System (Java)

## Description
This is a console-based Java application to manage student records using arrays and basic OOP concepts.

## Features
- Add Student
- Display Students
- Search Student by ID

## Technologies Used
- Java
- Arrays
- Object-Oriented Programming

## How to Run
1. Compile:
   javac Main.java StudentManagement.java
2. Run:
   java Main

## Sample Output
Enter your choice: 1  
Enter Student ID: 101  
Enter Student Name: Ibrahim  
Student added successfully  

## Concepts Used
- Classes & Objects
- Arrays
- Loops
- Conditional Statements
- Linear Search
